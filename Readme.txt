1. https://developer.android.com/studio/releases/platform-tools?hl=ko 접속
2. 각 운영체제에 맞는 SDK 플랫폼 다운로드
3. 압축풀어 최상단 폴더 이름이 platform-tools가 되도록 폴더 이동
4. 마켓에서 App Inspctor 설치하고 실행
5. 지울 앱 선택
6. Package name 밑에 있는 영어를 Database.txt에 입력
7. 2개 이상 지울 경우에는 한줄 띄우기(\n)로 구분하여 입력

ex)
com.android.vending
com.google.android.projection.gearhead
